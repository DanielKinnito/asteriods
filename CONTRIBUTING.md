# Contributing to Asteroids

Thank you for your interest in contributing to the Asteroids project!

## How to Contribute

1.  **Fork the repository.**
2.  **Create a new branch** for your feature or bugfix.
3.  **Make your changes**. Ensure your code is clean and readable.
4.  **Test your changes** by running the game.
5.  **Submit a Pull Request** describing your changes.

## Development Setup

1.  Install Python 3.13+.
2.  Install dependencies: `uv sync` or `pip install -r requirements.txt`.
3.  Run the game: `uv run main.py`.

## Style Guide

- Follow PEP 8 guidelines for Python code.
- Keep classes and functions focused and modular.
- Add comments for complex logic.
